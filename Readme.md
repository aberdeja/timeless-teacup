#Timeless Teacup

This is a website that is meant to represent an online storefront for a fake product. The Timeless Teacup in particular is designed to represent a shop that sells different types of teas. This website utilizes a combination of HTML and CSS in it's design. The goal of this website is to be made available on any browser. However, at the moment, the most compatible browser is Chrome.

#About Us

The about page contains a brief blurb regarding how the business started up.

#Contact Us

The contact page contains a form that allows users to contact the owners of the website. The form is designed using html and css.

#Products

The products page contains a list of products that are offered at the Timeless Teacup.

#Register

The register page contains a javascript form that allows users to sign up for the Timeless Teacup website. This javascript form features form validation. It ensures users include a lowercase letter, a capital letter, a number, and at least 8 characters in their password. It also ensures that users enter a valid email.
